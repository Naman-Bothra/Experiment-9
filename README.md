### Aim : Study of Numpy library in python

### Theory :
NumPy (Numerical Python) is a powerful open-source library used in Python for numerical and mathematical computations. It provides support for large, multi-dimensional arrays and matrices along with a collection of mathematical functions to operate on these data structures efficiently.

The core object of NumPy is the ndarray (N-dimensional array). Unlike Python lists, NumPy arrays store elements of the same data type and are more memory-efficient and faster for numerical operations. NumPy arrays allow vectorized operations, meaning mathematical computations can be performed on entire arrays without using explicit loops, which improves performance.

NumPy provides various functionalities such as:

Array Creation: Arrays can be created using functions like array(), zeros(), ones(), arange(), and linspace().

Array Operations: Arithmetic operations (addition, subtraction, multiplication, division) can be performed element-wise between arrays.

Indexing and Slicing: Elements or subsets of arrays can be accessed and modified using indexing and slicing techniques.

Shape Manipulation: Functions like reshape(), flatten(), and transpose() help in changing the structure of arrays.

Mathematical Functions: NumPy provides built-in mathematical operations such as mean, sum, standard deviation, square root, and trigonometric functions.

Multi-Dimensional Arrays: It supports 1D, 2D, and higher-dimensional arrays, which are widely used in data analysis, machine learning, and scientific computing.

## Commands Used :

a.shape :  Tells us about the shape of matrix/array

a.dtype : Tells us about the datatype of matrix/array

np.zeros : makes a null matrix

np.ones : makes an matrix of 1's

np.eye : makes an identity matrix

np.arange : used to create an array of evenly spaced values within a given range.

np.linspace : creates an array of evenly spaced numbers between two specified values.

np.mean : mean of matrix

np.median : median of matrix

np.min : displays the minimum value of a matrix

np.max : displays the maximum value of a matrix

np.sum : Displays sum of matrix
